### Dự án game nhỏ làm việc với thư viện SFML, tôi làm dự án nhỏ này với các mục đích:
- Học cách làm game bằng SFML
- Học cách quản lí cấu trúc thư mục của dự án. Vì nếu nhìn mấy repo trước của tôi thì như cái nồi cám lợn, nên tôi quyết tâm học về kĩ năng này
- Học cách dùng các lệnh terminal, g++,... (tránh lạm dụng IDE)
- Học cách quản lí mã nguồn, thay vì dồn một đống vào main.cpp thì sẽ chia ra nhiều tệp header và implementation, có thể nói là đang học cách áp dụng OOP
- Học thêm cách dùng git

### CÁCH BUILD VÀ CHẠY:
- Chạy các lệnh sau trong thư mục của dự án  
(quá trình này có thể mất hơn 2 phút)
```powershell
cmake -B build
cmake --build build
```
